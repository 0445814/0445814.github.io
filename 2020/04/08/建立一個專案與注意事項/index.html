[NuxtJS] Nuxt 教學(1) - 建立一個專案與注意事項

發表於 2020-04-08 分類於 NuxtJS
建立你的 Nuxt 專案吧~!

起手式

如同 Vue-Cli，Nuxt 也提供腳手架工具讓開發者快速建立一個專案。請先確認自己有安裝 Node.js，才能執行下載哦 !

npx 命令會在執行後，將套件刪除，這樣就可避免在本地電腦多一些不必要的檔案而污染到全域套件。

專案設定

開啟 Nuxt 專案

使用編輯器，路徑選擇建立的專案，並使用命令執行。

Nuxt 提供幾種模式

  • 開發模式

    • npm run dev

  • 生產模式

    • npm run build (使用 webpack 將資源打包，部屬用)
    • npm run start (以生產模式開啟 web 伺服器)

  • 靜態應用

    • npm run generate

執行 npm run dev 後，打開預設的 3000 port，就可以看到:

注意事項

在 Nuxt 中，必須注意引入的套件或撰寫時是否使用到瀏覽器的 API，例如 alert()，因為 Nuxt 底層的 Node 是不能使用瀏覽器語法的。例如我在 pages/index.vue 中加入一個 created 生命週期並寫 alert()

出錯了 :(

撰寫 Nuxt 時，要考慮到這是 Node.js 執行環境，Node 是不支援瀏覽器 window 屬性的。
