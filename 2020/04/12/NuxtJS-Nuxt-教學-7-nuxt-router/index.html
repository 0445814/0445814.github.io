[NuxtJS] Nuxt 教學(7) - nuxt-router

發表於 2020-04-12 分類於 NuxtJS

終於來到 nuxt-router 了，router 的配置也是我認為 Nuxt 強大的地方，只需簡單步驟就可建立路由

建立路由

Nuxt 會監聽 pages 的目錄結構變動來自動配置路由，例如在 pages 新增 user 目錄並新增 index.vue 與 login.vue

Nuxt router 將會自動生成路由，可以在 .nuxt/router.js 中找到

注意: name 的命名是由 Nuxt 規定的
pages 的 index.vue 默認首頁 /

目錄名稱下的 index.vue 將默認是該路由的頁面。

動態路由

在 Nuxt 中只需要使用下底線 _ 作為路由名稱前綴，就能建立動態路由

:id? 後面的問號表示路由是可選的，想將它設置為必選的路由，需要在 products 目錄內建立一個 index.vue 即可。

一般來說，我們可以透過 this.$route 取得動態路由參數對後端發請求。接著，取得回應資料後再渲染到畫面上。

路由參數驗證

Nuxt 可以讓你在動態路由組件中定義參數驗證方法。

如果驗證方法返回的值不為 true 或 Promise 中 resolve 解析為 false 或拋出 Error ， Nuxt 將自動加載顯示 404 錯誤頁面或 500 錯誤頁面。

更多驗證方法

嵌套路由

建立內嵌子路由，只需新增一個 Vue 文件，同時添加一個與該文件同名的目錄用來存放子視圖組件。

注意，父組件中記得使用 <nuxt-child> 來顯示子組件

跳轉路由

這與 Vue 的 router-link 相同，Nuxt 中使用 nuxt-link，即可跳轉路由

router-link 與 nuxt-link 差別，在這篇講過。
