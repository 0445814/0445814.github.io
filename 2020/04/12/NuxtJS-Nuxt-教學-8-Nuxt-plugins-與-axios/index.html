[NuxtJS] Nuxt 教學(8) - Nuxt plugins

發表於 2020-04-12 分類於 NuxtJS

Nuxt 引入插件的方式與 Vue-cli 稍有不同，這也是初學 Nuxt 容易發生錯誤的地方，來看看如何在 Nuxt 安裝套件吧。

Vue 中安裝套件

  • 使用 script 標籤引入 CDN 或 JS 檔案
  • 使用 npm install 安裝一般 JS 套件，再 import 引入
  • Vue-cli 中以 npm install 安裝 Vue 套件，再以 Vue.use 或 Vue.component 註冊

Nuxt 中安裝套件

  • nuxt.config.js 中以 CDN 方式引入 (較簡單)
  • 使用 nuxt plugins 安裝 (較麻煩)

Nuxt 在網頁第一次載入時，會使用 server side render 在後端執行，所以撰寫 Nuxt 時，要時時刻刻考慮 Node 執行環境。

而 nuxt.config.js 的 plugins mode，可以讓套件只在 client 端執行，這樣就能正確使用套件了。
